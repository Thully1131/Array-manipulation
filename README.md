# Array-manipulation

- Run the index.html file- if using Visual Studio Code open it with a live server.
- You can input a name in the input field and click the add student button and that name will be added to the empty array.
- The name will be displayed on the web page along with it's corresponding remove, edit and save button.
- If you want to remove a name in the array, just click the remove button.
- If you want to edit or change a name in the array, just click the edit button and that name will pop up on the input field.
- You can then change/edit the name in the input field and click the save button to save the new name in the array.
